🔁 The Loop

Six phases, seven seats, one round. Propose → Hold-apart → Assay runs as a pipeline with no barrier; the Critic is the only barrier, because classifying the round needs the whole round. After K consecutive dry rounds the loop stops rather than grinding — and a round that loses a seat is FAILED, not dry: an outage is never an exhausted search.

The seats, in phase order

  • 📐 Seat — Measure — frontier numbers only, no advocacy.
  • 🧙 Seat — Propose Soulbae — blind lenses in parallel; the complement, bnot.
  • ⿻ Seat — Hold-Apart The Gap — hashes the proposal to draw witnesses neither side chose; xor.
  • ⚔️ Seat — Assay Soulbis — the full gate; the negation, neg. Verdicts: VALIDATED · MIRAGE · BLOCKED.
  • 🧪 Seat — Critic — classifies the round: structural · probe-limited · noise · mis-gated (the only class that accuses the config, not the proposer).
  • 📜 Seat — Chronicle — drafts the record, verdict first.
  • 🗝️ Seat — Keystone — the pair itself in the main session: folds validated levers, writes the ledgers, and stops at the door.

What the loop never does

It never folds its own wins (the keystone's job), never writes the ledgers (keystone-only, GR-10), and never opens the door — push, publish, submit belong to the First Person alone (T6). See The Ten Ground Rules and The Six Trusts.
