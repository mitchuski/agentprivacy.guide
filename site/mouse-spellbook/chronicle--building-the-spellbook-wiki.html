Building the Spellbook Wiki

How the mouse_spellbook kit became this wiki on Mouse Spellbook. 2026-06-22.

What was built

73 pages: the on-ramp, the seven-step command spine (The Workshop → Step 1 — Copy the Narrative … Step 7 — Weave the Mouse Book), the full 31-act canon (The 31 Acts), the reference shelf, The Templates, and these The Chronicles.

How

A generator (~/.wiki/_gen_spellbook.js) reads the source tree and writes one FedWiki page per file — a markdown body plus a footer nav item (↑ up · ← prev · next → · See also), prev/next chained per group so a reader is guided forward and back. Backticked file refs became real wiki links; the narratives' own bracket device (RPP blocks, spoken proverbs) was neutralised to literal brackets so only true navigation links exist. A validator (~/.wiki/_validate_spellbook.js) confirmed every link resolves — 0 dangling, 0 orphans.

Coherence

The reference packet still labelled the story as 18 acts (and itself held fewer); the canon is complete at 31. The wiki was made coherent at 31 — The 31 Acts is authoritative — and The Narrative Packet now carries a note that it is an abridged rendering.

Split-ready

The generator's HOST_OF_GROUP map keeps everything on Mouse Spellbook for now; flipping it splits the canon to privacymage, the chronicles to chronicle, and a site per mouse later.
