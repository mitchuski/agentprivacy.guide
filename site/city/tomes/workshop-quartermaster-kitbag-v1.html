Ceremony Shape — DRAW · FIT · RETURN

  • DRAW. Take the standard skeleton — the six-phase loop, the seven seats, the conformance gate. Unforked, whole.
  • FIT. Lap the constellation; write your config onto the rig — which target, which seats, which trusts. Never fork the frame.
  • RETURN. The rig crystallises to your working. When done, the frame returns unchanged, so the next mage draws the same true rig.
