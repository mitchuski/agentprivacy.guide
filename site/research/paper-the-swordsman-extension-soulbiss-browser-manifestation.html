Abstract

The Swordsman Extension (codenamed swordsman-blade) is the browser manifestation of Soulbis, the Master agent in the dual-agent sovereignty architecture. Operating on the right side of the sovereignty interface, the Swordsman deals in assertion — deliberate stances, privacy postures, and blade forging for sovereignty declaration.

This document describes the Swordsman Extension's architecture, stance system, blade forge, control scheme, and canvas ownership responsibilities.

The Master's Role

In the dual-agent architecture, the Swordsman (Soulbis) handles:

  • Protection — Guarding the private ledger, making boundary decisions
  • Assertion — Declaring privacy postures through deliberate stances
  • Observation — Scanning pages for privacy threats and gap scores
  • Canvas Ownership — Rendering both orbs when Mage is present

The Swordsman OWNS the canvas overlay. When the Mage is present, rendering data flows TO the Swordsman, who renders a unified dual-orb view.

Gap Reduction System

The Swordsman's primary function is reducing the gap score — the measure of privacy vulnerability on a page.

Gap Score Calculation

Gap Score = Σ (threat_weight × threat_count)

Where threats include:
- Third-party trackers
- Tracking cookies
- Dark patterns
- PII collection forms
- Surveillance scripts

Stance Impact Formula

function calculateGapReduction(
  stance: SwordsmanStance,
  currentGap: number
): number {
  // Base reduction from stance weight
  let reduction = stance.weight * 2

  // Blocking stances have stronger effect on high-gap pages
  if (isBlockingAssertion(stance.myTermsMapping)) {
    reduction += Math.floor(currentGap / 20)
  }

  // Sovereign stance has maximum effect
  if (stance.myTermsMapping === 'FULL_SOVEREIGNTY') {
    reduction += 5
  }

  return Math.min(reduction, currentGap)
}

Spring Physics

The Swordsman orb uses spring physics for natural movement:

Tether System

class SpringTether {
  x: number
  y: number
  vx: number = 0
  vy: number = 0

  // Spring constants
  stiffness: number = 0.02
  damping: number = 0.9

  update(targetX: number, targetY: number) {
    // Spring force toward target
    const dx = targetX - this.x
    const dy = targetY - this.y

    this.vx += dx * this.stiffness
    this.vy += dy * this.stiffness

    // Damping
    this.vx *= this.damping
    this.vy *= this.damping

    // Update position
    this.x += this.vx
    this.y += this.vy
  }
}
