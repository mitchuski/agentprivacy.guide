💡 The Holon Layer (κ as state)

concept · shared · knowledge

The dual-agent harness's content-addressing substrate (HOLONS.md 2026-07-12): every result it mints — artefact, runtime-feed, verdict — carries a κ-label, κ = sha256(canonicalJson(object with the κ field removed)), the City Key / sigil Law L5 unchanged (a κ-label is never trusted, only re-derived). A holon is a whole-that-is-also-a-part: it stands alone and composes into a mesh by κ→κ edges — content edges re-hash (the content is its own witness), relational edges (the VRC) need the counterparty's signature to mint. tools/kappa.mjs is the SINGLE shared κ primitive (one function everywhere so addressing never drifts — the opposite of the independent-copy idiom used for gate logic); tools/holon_audit.mjs is a deterministic, agent-free AUDITOR of the mesh. The division is total: the harness mints the holons, the auditor verifies the mesh. The interoperability substrate hologram-technologies seats on.

A κ-label is never trusted, only re-derived.
